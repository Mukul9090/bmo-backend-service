# GitHub Setup Instructions

Your repository is ready! Follow these steps to push to GitHub:

## Step 1: Create GitHub Repository

1. Go to https://github.com/new
2. Repository name: `bmo-backend-service` (or your preferred name)
3. Description: "Flask backend service with Kubernetes deployment for high availability demo"
4. Choose: **Public** or **Private**
5. **DO NOT** initialize with README, .gitignore, or license (we already have these)
6. Click "Create repository"

## Step 2: Add Remote and Push

After creating the repository, GitHub will show you commands. Use these:

```bash
# Add the remote (replace YOUR_USERNAME with your GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/bmo-backend-service.git

# Or if you prefer SSH:
# git remote add origin git@github.com:YOUR_USERNAME/bmo-backend-service.git

# Push to GitHub
git branch -M main
git push -u origin main
```

## Alternative: Using GitHub CLI

If you have GitHub CLI installed:

```bash
# Create repo and push in one command
gh repo create bmo-backend-service --public --source=. --remote=origin --push
```

## Verify

After pushing, check your repository:
```bash
git remote -v
```

You should see your GitHub repository URL.
